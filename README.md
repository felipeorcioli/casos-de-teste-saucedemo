# Casos de Teste — Saucedemo

Documentação de casos de teste e bug reports do site [Saucedemo](https://www.saucedemo.com), 
um e-commerce fictício utilizado para prática de QA.

## 📁 Estrutura

| Pasta | Conteúdo |
|---|---|
| `/login` | Casos de teste da funcionalidade de login |
| `/bugs` | Bug reports dos bugs encontrados durante testes exploratórios |

## 🧪 Resumo

| Tipo | Quantidade |
|---|---|
| Casos de teste | 3 |
| Bug reports | 4 |

## 🛠️ Ferramentas utilizadas
- Testes exploratórios manuais
- Documentação em Markdown
